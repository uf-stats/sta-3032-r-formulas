Module 1: Summary and Descriptive Statistics

num1 and num2 represent two vectors of data

  • mean(num1, trim=0.05) where (0.05 means 5% trim)
  • quantile(num1,0.25,type=6) where (0.25 means 25th percentile)

Module 2: Probability and Random Variables

Couting

  • factorial(x)
  • choose(n, x)

Discrete Distributions

  • Binomial
    • dbinom(x, size, prob)
    • pbinom(x, size, prob)
    • qbinom(p, size, prob)
    • x = number of successes, size = number of trials, prob = probability of success on each trial, p = percentile (as a decimal)
  • Geometric
    • dgeom(x, prob)
    • pgeom(x, prob)
    • qgeom(p, prob)
    • x = number of failures, prob = probability of success on each trial, p = percentile (as a decimal)
  • Negative Binomial
    • dnbinom(x, size, prob)
    • pnbinom(x, size, prob)
    • qnbinom(p, size, prob)
    • x = number of failures, size = target number of successes, prob = probability of success on each trial, p = percentile (as a decimal)
  • Poisson
    • dpois(x, lambda)
    • ppois(x, lambda)
    • qpois(p, lambda)
    • x = number of occurrences, lambda = fixed rate, usually per unit of time, p = percentile (as a decimal)

Continuous Distributions

  • Uniform
    • punif(x, a, b)
    • qunif(p, a, b)
    • x = random variable value, a = lower boundary point, b = upper boundary point, p = percentile (as a decimal)
  • Normal
    • pnorm(x, mu, sigma)
    • qnorm(p, mu, sigma)
    • x = random variable value, mu = mean, sigma = standard deviation, p = percentile (as a decimal)
  • Chi-Square
    • pchisq(x, df)
    • qchisq(p, df)
    • x = random variable value, df = degrees of freedom, p = percentile (as a decimal)
  • t-Distribution
    • pt(x, df)
    • qt(p, df)
    • x = random variable value, df = degrees of freedom, p = percentile (as a decimal)
  • F-Distribution
    • pf(x, df1, df2)
    • qf(p, df1, df2)
    • x = random variable value, df1 = degrees of freedom, df2 = degrees of freedom, p = percentile (as a decimal)

Modules 3 and 4: Inference for One and Two Populations

Inference for Population Means (one or two samples)

  • t.test(x, y = NULL, alternative="two.sided", mu = 0, paired = FALSE, var.equal = FALSE, conf.level = 0.95)
    • x = vector of data values
    • y = optional vector of data values (two sample only)
    • alternative = string specifying alternative ("two.sided", "greater", "less")
    • mu = null value mean
    • paired = TRUE if paired data (two sample only)
    • var.equal = TRUE if assume equal variances (two sample only)
    • conf.level = confidence level as a decimal (enter 0.95 for 95%)

Inference for Population Proportion (one sample only)

  • binom.confint(x, n, conf.level = 0.95, methods = "all")
    • Need to load "binom" package
    • x = number of successes in the binomial experiment
    • n = number of trials in the binomial experiment
    • conf.level = confidence level as a decimal (enter 0.95 for 95%)
    • methods = methods used to construct interval(default is "all", "ac" provides Agresti-Coull)

Inference for Population Variances (one or two samples)

  • varTest{library(EnvStats)}(x, y = NULL, alternative = "two.sided", sigma.squared = 1, conf.level = 0.95)
    • x = vector of data values
    • y = optional vector of data values (two sample only)
    • alternative = string specifying alternative ("two.sided", "greater", "less")
    • sigma.squared = null value variance
    • conf.level = confidence level as a decimal (enter 0.95 for 95%)

Sign Test (one sample only)

  • eqnpar(x, p = 0.5, type = 6, ci = TRUE, ci.method = "exact", ci.type = "two-sided", approx.conf.level = 0.95)
    • Need to load "EnvStats" package
    • x = vector of data values
    • p = percentile as a decimal (enter 0.5 for 50th percentile)
    • type = calculation method, use type = 6 for this class
    • ci = FALSE means do not compute confidence interval
    • ci.method = method to computing confidence interval, use ci.method = "exact" for this class
    • ci.type = string specifying type ("two-sided", "lower", "upper")
    • approx.conf.level = confidence level as a decimal (enter 0.95 for 95%)

Wilcoxon Signed-rank Test (one or two samples)

  • wilcox.test(x, y = NULL, mu = 0, paired = FALSE, alternative = "two.sided", conf.int = TRUE, conf.level = 0.95)
    • x = vector of data values
    • y = optional vector of data values (two sample only)
    • mu = null value mean
    • paired = TRUE if paired data (two sample only)
    • alternative = string specifying alternative ("two.sided", "greater", "less")
    • conf.int = FALSE means do not compute confidence interval
    • conf.level = confidence level as a decimal (enter 0.95 for 95%)

Chi-Square Test (two samples only)

  • chisq.test(x)
    • x = matrix of data values

Module 5: Linear Regression

Simple Linear Regression Model

  • reg = lm(y ~ x, data = myData)
    • y = dependent (outcome/response) variable
    • x = independent (predictor) variable
    • data = data frame that includes columns of y and x

Multiple Linear Regression Model

  • reg = lm(y ~ x1 + x2 + x3, data = myData)
    • y = dependent (outcome/response) variable
    • x1 = first independent (predictor) variable
    • x2 = second independent (predictor) variable
    • x3 = third independent (predictor) variable (if necessary)
    • data = data frame that includes columns of y, x1, x2, etc

Inference for Linear Regression

  • summary(reg)
  • anova(reg)
  • confint(reg, level = 0.95)
    • reg = linear regression model
    • level = confidence level as a decimal (enter 0.95 for 95%)
  • predict.lm(reg, newdata = data.frame(x = Predictor), interval = "confidence", level=0.95)
    • reg = linear regression model
    • newdata = data frame of values of predictor, x. If multiple predictors, they must be entered separately (for example x1 = x1.data, x2 = x2.data)
    • interval = string specifying whether a confidence interval ("confidence") or a prediction interval ("prediction")
    • level = confidence level as a decimal (enter 0.95 for 95%)

Box-Cox

  • bc = powerTransform(y ~ x, data = myData)
    • y = dependent (outcome/response) variable
    • x = independent (predictor) variable
    • data = data frame that includes columns of y and x
  • summary(bc)

Module 6: Analysis of Variance

Fitting and Inference:

  • reg = aov(y ~ x, data = myData)
    • y = dependent (outcome/response) variable
    • x = independent (predictor) variable
    • data = data frame that includes columns of y and x
  • reg = aov(y ~ x + block, data = myData)
    • y = dependent (outcome/response) variable
    • x = independent (predictor) variable
    • block = block variable
    • data = data frame that includes columns of y, x, and block
  • summary(reg)
  • anova(reg)

Multiple Compairsons

  • TukeyHSD(reg)
